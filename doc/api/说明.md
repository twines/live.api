# 前端接口默认的路由前缀/api/v1

## 接口应该先判断HTTP状态码
## 再判断接口状态码
## 最后取data里的数据
## data可能是map也可能是list

### 成功示例
```
{
    "code": 20000,
    "msg": "success",
    "data": []
}

```

### 失败示例
```
{
    "code": 40000,
    "msg": "用户已经存在"
}

```

##HTTP状态码说明：
|状态码|说明|备注|
|:---:|:---:|:---:|
|403|禁止||
|200|成功||

##接口状态码说明：
|状态码|说明|备注|
|:---:|:---:|:---:|
|20000|成功||
|40000|失败||
